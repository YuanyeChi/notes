Java在编译过程中不存在未初始化的变量。

所有数据都应该是私有的。所有类的数据都应该在类的内部改。`set()`,`get()`

- package
package必须是程序的第一行，package abc.a.b.c,代表abc下的子文件夹a下的子文件夹b下的子文件夹c

- static
静态的变量就是所有类实例化出来的对象共享变量。类的变量/方法，独立于类的对象，可以根据类名直接调用。

- 单类模式
所有的东西都在一个类里面，所以一个类中必须有main函数
- 多类模式
有一个主控类，负责管控其他类的调用。

- 成员方法
在C系列的函数传递的都是值，但是C++中传递的是指针或者引用。Java中也是传值的过程。
- 构造方法
构造方法是不能有返回值的
```java
class test {
	int a, b;//成员变量
	test (int a, int b) {//局部变量
		this.a = a;//加了this就是调用类的成员变量
		this.b = b;
	}
}
```

- 方法的重写(override)
存在与父类和子类之间
- 辨析
